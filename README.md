# ATM System (Python Project)
A beginner-friendly Python console-based ATM simulation system that allows users to perform basic banking operations like checking balance, depositing money, withdrawing money, and transferring funds with PIN authentication and transaction tracking.

# Features
• PIN-based authentication system (3 attempts limit)  
• Check account balance  
• Deposit money into account  
• Withdraw money with validation checks  
• Transfer money to another account  
• Transaction history tracking  
• Receipt generation for each transaction  
• Input validation for invalid amounts  
• Account lock after multiple failed PIN attempts  

# Concepts Used
• Variables  
• Functions  
• Loops (while loop)  
• Conditional statements (if-elif-else)  
• Lists (transaction history)  
• User input handling  
• Basic error handling  
• Arithmetic operations
